## Installation

### Default Installation

#### Install Manually

Add this to your `Cargo.toml`:
```toml
[dependencies]
mod-cli = "0.5.0"
```

<br>

#### Install via Terminal
```bash
# Basic installation
cargo add mod-cli
```
